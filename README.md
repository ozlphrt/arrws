# ARROWS

A Progressive Web App with a canvas displaying 36×18 dots (portrait layout), built with Vite + React. Ready for snake animations with touch interactions.

## Tech Stack

- **Vite** - Fast dev server and build tool
- **React** - Component-based UI for managing animations and state
- **Canvas API** - Efficient rendering of dots and future snake animations
- **PWA** - Installable with offline support via Vite PWA plugin

## Features

- **36×18 dot grid** - Portrait layout, responsive canvas
- **Touch/Tap handlers** - Ready for snake animation triggers
- **PWA ready** - Installable on mobile devices
- **Offline support** - Service worker auto-generated by Vite PWA plugin
- **Mobile optimized** - Touch-friendly, responsive design
- **High-DPI support** - Crisp rendering on retina displays

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start dev server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```
